# Pontaj Angajati

Genereaza automat pontajul la o persoana tinand cont de zilele libere legale.

### Cum pornesc aplicatia

    $ npm install
    $ npm run dev

### Demo

[https://pontaj-angajati.netlify.app](https://pontaj-angajati.netlify.app)

### Cum functioneaza

    - Introdu numele angajatului
    - Seteaza luna dorita
    - Seteaza anul dorit
    - Print
